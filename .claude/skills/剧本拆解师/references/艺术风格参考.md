# 艺术风格参考

## 常见2D游戏美术风格

### 像素艺术
- **特点**：低分辨率像素点构成，复古感强
- **代表作品**：《星露谷物语》、《时空之轮》
- **适用场景**：复古游戏、独立游戏、休闲游戏
- **资源特点**：需要明确像素尺寸和调色板

### 手绘风格
- **特点**：手绘笔触，艺术感强
- **代表作品**：《空洞骑士》、《茶杯头》
- **适用场景**：强调叙事和美术的游戏
- **资源特点**：需要详细的线稿和上色说明

### 扁平化风格
- **特点**：简洁图形，纯色块
- **代表作品**：《纪念碑谷》、《Alto's Adventure》
- **适用场景**：休闲游戏、解谜游戏
- **资源特点**：需要明确的配色方案和几何形状

### 卡通风格
- **特点**：夸张造型，鲜艳色彩
- **代表作品**：《植物大战僵尸》、《愤怒的小鸟》
- **适用场景**：休闲游戏、儿童游戏
- **资源特点**：需要明确的角色设定和表情变化

### 写实风格
- **特点**：接近真实视觉效果
- **代表作品**：《地狱边境》、《Inside》
- **适用场景**：叙事游戏、恐怖游戏
- **资源特点**：需要高精度素材和光影效果

## 风格描述要点

当用户指定某种风格时，需要确认：

| 要点 | 说明 |
|------|------|
| 参考作品 | 用户是否有参考游戏 |
| 色彩方案 | 主色调、配色风格 |
| 线条处理 | 是否有轮廓线、线条粗细 |
| 质感表现 | 是否需要纹理、光影效果 |
| 尺寸规范 | 资源的标准尺寸 |
| 动画风格 | 帧动画、骨骼动画、形变动画 |
