## 核心玩法

### 操作方式

| 按键 | 功能 |
|:----:|------|
| `W` `A` `S` `D` | 移动 |
| `空格` | 翻滚闪避 |
| `左键` | 普通攻击 / 射击 |
| `右键` | 终极技能 |
| `Q` | 技能 1 |
| `E` | 技能 2 |
| `R` | 换弹 / 冥想（法师恢复魔法） |
| `F` | 拾取物品 |

> 鼠标指向即为目标，左右键对目标执行操作。

### 核心机制

1. **搜索物资**：探索废土场景，收集资源和装备
2. **战斗系统**：即时战斗，支持近战/远程/魔法攻击
3. **角色强化**：通过战斗获取经验与资源，提升角色能力
4. **剧情推进**：双主角叙事，探索核战后世界的秘密
5. **副本挑战**：房间式副本结构，包含普通/精英/Boss/隐藏房间

### 经济循环

```
完成任务 ──→ 获得货币
    │
    ↓
击杀怪物 ──→ 获得怪物精粹 ──→ 商人兑换货币
    │
    ↓
击杀人形敌人 ──→ 直接获得货币
    │
    ↓
货币 ──→ 购买物品/装备/服务
```

## 世界观概述

核战之后，世界支离破碎。幸存的人类分化为两大阵营：

| 阵营 | 特征 |
|------|------|
| **科技派** | 依赖科技与机械力量 |
| **魔法派** | 身体变异，可释放魔法 |

**冲突根源**：资源匮乏，两大阵营为争夺生存资源而持续争斗。

| 魔法主角 | 魔法派·秘术协会 | 女，120岁（魔法派寿命可达300岁）。14年前任务中误杀男主双亲 |

## 势力阵营

### 科技派

| 组织 | 设计主题 | 作战风格 |
|------|:--------:|----------|
| 金属合唱团 | 深红 | 远程枪械攻击 |
