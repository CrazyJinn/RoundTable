## 核心玩法

### 操作方式
| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| `空格` | 翻滚（闪避） |
| `左键` | 普通攻击 / 普通射击 |
| `右键` | 终极技能 |
| `R` | 换子弹（科技派）/ 冥想（魔法派，冥想后可释放N次魔法） |
| `Q` | 技能1 |
| `E` | 技能2 |
| `F` | 拾取 / 交互 |

> 鼠标的焦点就是目标，按左右键就是对目标进行操作（射击、使用技能）

### 核心机制
1. **双阵营系统**：玩家可选择科技派或魔法派，两派拥有截然不同的战斗风格和能力
2. **资源争夺**：在废土世界搜索物资，用于强化技能和购买装备
3. **战斗系统**：俯视角即时战斗，支持近战/远程/召唤多种作战方式
4. **翻滚闪避**：通过翻滚躲避敌人攻击，增加战斗策略性

## 世界观概述

核战后，世界支离破碎，幸存的人类分成了两派：

- **科技派**：依靠科技力量生存，包括使用枪械、外骨骼机甲、机器人等
- **魔法派**：因核辐射发生身体变异，获得释放魔法的能力，寿命可达300岁

资源有限，两派人经常为了争抢资源而互相争斗。

## 势力阵营

### 科技派

| 组织 | 设计主题 | 作战风格 |
|------|----------|----------|
| 金属合唱团 | 深红 | 远程攻击为主，以枪械作为攻击手段 |
| 骑士团 | 黑白灰 | 近战攻击为主，穿着外骨骼机甲 |
| 机械风暴 | 待定 | 以召唤物（机器人、无人机）作战 |

### 魔法派

| 组织 | 设计主题 | 作战风格 |
|------|----------|----------|
| 教廷 | 金色、白色 | 近战攻击为主，靠祷告强化自身 |
| 秘术协会 | 彩虹色 | 远程攻击为主，释放杀伤性魔法 |
