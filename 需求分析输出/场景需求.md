## 场景详细需求

### map_001 - 中央废土

#### 基本信息
| 项目 | 内容 |
|------|------|
| 名称 | 中央废土 |
| 类型 | 大地图 |
| 描述 | 核战后的荒芜之地，散布着废墟和辐射坑，是游戏的主要探索区域 |
| 氛围 | 白天/黄昏/夜晚 |

#### 视觉需求
| 资源类型 | 规格 | 数量 | 说明 |
|----------|------|------|------|
| 背景图 | 4096x4096 | 3张 | 不同时间段背景 |
| 瓦片/地块 | 64x64 | 20种 | 沙地、碎石、裂缝地面、辐射坑等 |
| 物品/装饰 | 32x32~128x128 | 30种 | 废墟残骸、报废车辆、辐射标志、枯树等 |
| 特效 | 256x256 | 3种 | 辐射雾、沙尘、热浪 |

#### 图层结构
| 图层 | 内容 | 说明 |
|------|------|------|
| 天空层 | 阴沉天空、云层 | 最底层 |
| 远景层 | 远处废墟轮廓 | 视差滚动 |
| 中景层 | 主要场景元素 | 视差滚动 |
| 地面层 | 可行走区域 | 主图层 |
| 装饰层 | 废墟、残骸等 | 玩家可穿过 |
| 前景层 | 遮挡物 | 最前景 |

#### 交互元素
| 元素ID | 类型 | 功能 | 说明 |
|--------|------|------|------|
| interact_001 | 物资箱 | 拾取物品 | 随机刷新 |
| interact_002 | 传送点 | 场景切换 | 连接各区域 |
| interact_003 | 怪物刷新点 | 战斗触发 | 随机生成怪物 |

## UI界面详细需求

### ui_001 - 主菜单

#### 基本信息
| 项目 | 内容 |
|------|------|
| 名称 | 主菜单 |
| 类型 | 菜单界面 |
| 触发条件 | 游戏启动时 |

#### UI元素
| 元素 | 类型 | 位置 | 功能 |
|------|------|------|------|
| 游戏标题 | 图片 | 居中上方 | 显示游戏名称 |
| 开始游戏 | 按钮 | 居中 | 进入角色选择 |
| 继续游戏 | 按钮 | 居中 | 读取存档 |
| 设置 | 按钮 | 居中 | 打开设置面板 |
| 退出 | 按钮 | 居中 | 退出游戏 |
| 背景图 | 图片 | 全屏 | 废土主题背景 |

### ui_002 - 游戏HUD

#### 基本信息
| 项目 | 内容 |
|------|------|
| 名称 | 游戏HUD |
| 类型 | HUD |
| 触发条件 | 游戏进行中始终显示 |

#### UI元素
| 元素 | 类型 | 位置 | 功能 |
|------|------|------|------|
| HP条 | 进度条 | 左上角 | 显示生命值 |
| MP/弹药条 | 进度条 | HP条下方 | 显示魔法值/弹药 |

| 交互提示 | 文本 | 角色下方 | 显示可交互对象提示 |
