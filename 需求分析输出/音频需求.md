## 背景音乐详细需求

### bgm_001 - 主菜单

| 项目 | 内容 |
|------|------|
| 名称 | 主菜单 |
| 用途 | 主菜单背景音乐 |
| 情绪 | 荒凉、神秘、希望 |
| 风格 | 混合（电子+管弦） |
| 时长 | 120秒 |
| 循环点 | 15秒 |

**详细描述：**
开场以低沉的电子合成器铺底，营造核战后荒凉的氛围。中段加入管弦乐元素，象征人类文明的残余。结尾处加入一丝希望的旋律，暗示重建的可能性。整体氛围压抑但不绝望。

---

### bgm_002 - 废土漫步

| 项目 | 内容 |
|------|------|
| 名称 | 废土漫步 |
| 用途 | 中央废土探索 |
| 情绪 | 荒凉、紧张、探索 |
| 风格 | 氛围电子 |
| 时长 | 180秒 |
| 循环点 | 30秒 |

**详细描述：**
以风声和环境音效为基底，加入不规则的电子节拍，模拟废土的不确定性。偶尔插入金属碰撞的声音，暗示废墟的存在。节奏缓慢，给玩家探索的空间感。

---

### bgm_003 - 战斗激昂

| 项目 | 内容 |
|------|------|
| 名称 | 战斗激昂 |
| 用途 | 普通战斗 |
| 情绪 | 紧张、激烈、快节奏 |
| 风格 | 电子摇滚 |
| 时长 | 90秒 |
| 循环点 | 10秒 |
