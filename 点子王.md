# 游戏设定

## 世界观
- 核战后，世界支离破碎，幸存的人类分成了两派：科技和魔法
- 一部分人身体发生了变异，可以使用释放魔法
- 一部分人使用科技的力量
- 资源有限，两派人经常为了争抢资源而互相争斗

## 角色
- **科技主角** — 男，17岁，开局隶属骑士团，3岁时双亲被女主误杀，但是因其幼小只有记忆的片段，骑士团将其救下
- **魔法主角** — 女，120岁，开局隶属秘术协会，魔法派别的人因为变异寿命普遍能有300岁。14年前的一次任务中杀死了男主双亲

# 游戏概览

## 综述
- 2D俯视角动作游戏
- 搜索物资、战斗之后可以使用物资强化自身技能、购买物资
- 人物为手绘风格，地图背景是油画风格
- 场景分为：每个组织的城镇，大地图，战斗的小场景
- 大地图上分：废土、变异森林、城市废墟、雪地（但是因为核冬天，雪是灰色的）

## 主要操作
| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 移动 |
| `空格` | 翻滚 |
| `左键` | 普通攻击 / 普通射击 |
| `右键` | 终极技能 |
| `R` | 换子弹（魔法师冥想，冥想一次后可释放N次魔法） |
| `Q` | 技能1 |
| `E` | 技能2 |
| `F` | 拾取 |

> 鼠标的焦点就是目标，按左右键就是对目标进行操作（射击、使用技能）
