## 游戏概览

### 基本信息

| 属性 | 描述 |
|------|------|
| 类型 | 2D 俯视角动作游戏 |
| 美术风格 | 人物：手绘风格 / 地图：油画风格 |
| 核心玩法 | 搜索物资 → 战斗 → 强化角色 → 推进剧情 |

### 场景结构

```
大地图
├── 各组织城镇（安全区）
├── 废土
├── 变异森林
├── 城市废墟
└── 灰雪地（核冬天）
```

**副本结构**：每个副本包含若干小地图，部分小地图有精英怪，最终地图有 Boss。首次通关可推进剧情。

### 操作说明

| 按键 | 功能 |
|:----:|------|
| `W` `A` `S` `D` | 移动 |
| `空格` | 翻滚闪避 |
| `左键` | 普通攻击 / 射击 |
| `右键` | 终极技能 |
| `Q` | 技能 1 |
| `E` | 技能 2 |
| `R` | 换弹 / 冥想（法师恢复魔法） |
| `F` | 拾取物品 |

> 鼠标指向即为目标，左右键对目标执行操作。

## 世界观

核战之后，世界支离破碎。幸存的人类分化为两大阵营：

| 阵营 | 特征 |
|------|------|
| **科技派** | 依赖科技与机械力量 |
| **魔法派** | 身体变异，可释放魔法 |

**冲突根源**：资源匮乏，两大阵营为争夺生存资源而持续争斗。

## 经济系统

### 货币体系

- **通用货币**：小口径手枪子弹
- **背景设定**：大口径子弹（战斗用）仍有生产线制造，但小口径子弹已无法生产，成为稀缺货币

### 经济循环

```
┌─────────────────────────────────────────────────┐
│                    经济循环                       │
├─────────────────────────────────────────────────┤
│                                                 │
│   完成任务 ──→ 获得手枪子弹                       │
│       │                                         │
│       ↓                                         │
│   击杀怪物 ──→ 获得怪物精粹 ──→ 商人兑换子弹       │
│       │                                         │
│       ↓                                         │
│   击杀人形敌人 ──→ 直接获得子弹                   │
│       │                                         │
│       ↓                                         │
│   手枪子弹 ──→ 购买物品/装备/服务                │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 势力阵营

### 科技派

| 组织 | 设计主题 | 作战风格 |
|------|:--------:|----------|
| **金属合唱团** | 深红 | 远程枪械攻击 |
| **骑士团** | 黑白灰 | 外骨骼机甲，近战为主 |
| **机械风暴** | 待定 | 召唤机器人/无人机作战 |

### 魔法派

| 组织 | 设计主题 | 作战风格 |
|------|:--------:|----------|
| **教廷** | 金色/白色 | 祷告强化自身，近战为主 |
| **秘术协会** | 彩虹色 | 释放杀伤性魔法，远程为主 |
| **灵魂行者** | 银色/黑色 | 召唤兽作战 |

## 角色设定

| 角色 | 基本信息 | 背景故事 |
|------|----------|----------|
| **科技主角** | 男，17岁<br>隶属：骑士团 | 3岁时双亲被女主误杀，仅存片段记忆。被骑士团救下并抚养长大。 |
