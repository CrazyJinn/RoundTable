## 游戏概览

### 基本信息

| 属性 | 描述 |
|------|------|
| 类型 | 2D 俯视角动作游戏 |
| 美术风格 | 人物：手绘风格 / 地图：油画风格 |
| 核心玩法 | 搜索物资 → 战斗 → 强化角色 → 推进剧情 |

### 场景结构

```
大地图
├── 各组织城镇（安全区）
├── 废土
├── 变异森林
├── 城市废墟
└── 灰雪地（核冬天）
```

**副本结构**：每个副本包含若干小地图，部分小地图有精英怪，最终地图有 Boss。首次通关可推进剧情。

### 操作说明

| 按键 | 功能 |
|:----:|------|
| `W` `A` `S` `D` | 移动 |
| `空格` | 翻滚闪避 |
| `左键` | 普通攻击 / 射击 |
| `右键` | 终极技能 |
| `Q` | 技能 1 |
| `E` | 技能 2 |
| `R` | 换弹 / 冥想（法师恢复魔法） |
| `F` | 拾取物品 |

> 鼠标指向即为目标，左右键对目标执行操作。

## 世界观

核战之后，世界支离破碎。幸存的人类分化为两大阵营：

| 阵营 | 特征 |
|------|------|
| **科技派** | 依赖科技与机械力量 |
| **魔法派** | 身体变异，可释放魔法 |

**冲突根源**：资源匮乏，两大阵营为争夺生存资源而持续争斗。

## 经济系统

### 货币体系

- **通用货币**：小口径手枪子弹
- **背景设定**：大口径子弹（战斗用）仍有生产线制造，但小口径子弹已无法生产，成为稀缺货币

### 经济循环

```
┌─────────────────────────────────────────────────┐
│                    经济循环                       │
├─────────────────────────────────────────────────┤
│                                                 │
│   完成任务 ──→ 获得手枪子弹                       │
│       │                                         │
│       ↓                                         │
│   击杀怪物 ──→ 获得怪物精粹 ──→ 商人兑换子弹       │
│       │                                         │
│       ↓                                         │
│   击杀人形敌人 ──→ 直接获得子弹                   │
│       │                                         │
│       ↓                                         │
│   手枪子弹 ──→ 购买物品/装备/服务                │
│                                                 │
└─────────────────────────────────────────────────┘
```

### NPC 交互机制

**好感度系统**：
- 与 NPC 交互、完成任务可提升好感度
- 好感度等级：陌生 → 熟悉 → 友好 → 信任 → 亲密
- 好感度提升可解锁：
  - 特殊商品折扣
  - 独家物品
  - 额外任务线
  - NPC 背景故事

**对话系统**：

#### 对话类型

| 类型 | 触发方式 | 特点 |
|------|----------|------|
| **闲聊** | 主动对话 | 获取背景信息、世界观碎片 |
| **任务对话** | 接取/完成任务 | 有明确目标，奖励固定 |
| **剧情对话** | 剧情节点触发 | 不可跳过，影响故事走向 |
| **商店对话** | 进入交易界面 | 可讨价还价（好感度高时） |
| **秘密对话** | 特殊条件解锁 | 隐藏信息、独家任务 |

#### 对话选项机制

**选项类型**：
```
┌─────────────────────────────────────────┐
│  [友好] 礼貌回应      → 好感度 +2       │
│  [中立] 简短回答      → 好感度 +0       │
│  [激进] 质问/威胁     → 好感度 -2       │
│  [特殊] ████████      → 需要特定条件    │
└─────────────────────────────────────────┘
```

**选项影响**：
| 影响范围 | 说明 |
|----------|------|
| 好感度 | 每次对话选项影响当前 NPC 好感度 |
| 剧情分支 | 关键对话决定后续剧情走向 |
| 任务奖励 | 好感度影响任务奖励丰厚程度 |
| NPC 态度 | 好感度过低时 NPC 可能拒绝交互 |

#### NPC 情绪反应

| NPC 情绪 | 触发条件 | 效果 |
|----------|----------|------|
| 友好 | 好感度 ≥ 友好 | 商品 9 折、额外对话选项 |
| 信任 | 好感度 ≥ 信任 | 商品 8 折、独家物品、秘密任务 |
| 亲密 | 好感度 ≥ 亲密 | 商品 7 折、特殊剧情、战斗支援 |
| 冷淡 | 好感度 ≤ 陌生 | 商品原价、对话简短 |
| 敌对 | 好感度极低 | 拒绝交互、可能攻击 |

#### 对话解锁条件

| 解锁内容 | 条件 |
|----------|------|
| 隐藏对话选项 | 完成特定任务 / 持有特定物品 |
| NPC 背景故事 | 好感度达到信任 |
| 秘密任务线 | 好感度达到亲密 + 完成前置任务 |
| 势力内部情报 | 加入该势力 + 好感度友好 |
| 核战历史碎片 | 收集足够情报道具 |

#### 情报收集系统

- **情报道具**：日记残页、录音带、旧报纸等
- **收集奖励**：解锁 NPC 额外对话、揭示世界观
- **情报关联**：某些情报需要组合才能理解完整信息

```
示例：老汤姆的回忆
├── 残破日记（第1页）→ 解锁"核战当天"话题
├── 军用徽章       → 解锁"旧军队"话题
└── 全家福照片     → 解锁"家人"话题 → 触发隐藏支线
```
