# 角色艺术风格定义

## 整体风格
- **风格名称：** 像素风废土角色
- **风格类型：** 像素艺术（Pixel Art）
- **角色特点：**
  - 清晰的轮廓和色彩边界
  - 简洁但富有表现力的设计
  - 废土风格与魔法风格的融合
  - 科技派采用黑白灰配色为主
  - 魔法派采用紫黑配色为主

## 角色色彩方案

### 科技派配色（骑士团）
- **主色调：** #2C2C2C - 深灰色（机甲主体）
- **副色调：** #1A1A1A - 黑色（机甲暗部）
- **强调色：** #E8E8E8 - 白色（机甲高光细节）
- **标识色：** #8B0000 - 深红色（骑士团标识）

### 魔法派配色（秘术协会）
- **主色调：** #1A1A1A - 黑色（服装主体）
- **副色调：** #9B30FF - 紫色（魔法装饰、内衬、宝石）
- **强调色：** #E066FF - 亮紫色（魔法光辉效果）
- **背景色：** #4B0082 - 深紫色（魔法背景）

### 怪物配色
- **辐射怪物：** #8B7355 - 灰褐色（变异生物）
- **血肉色：** #CD5C5C - 暗红色（伤口、血迹）
- **毒液色：** #32CD32 - 绿色（毒素、毒液）
- **骨白色：** #F5F5DC - 米白色（骨化部位）

## 角色设计规范
- **头身比例：** 3-4头身（适合像素风格）
- **线条风格：** 像素化硬边，无抗锯齿
- **质感描述：** 像素块状质感，高对比度
- **阴影处理：** 使用纯色块表现阴影，无渐变

## 角色尺寸规范

### 主角
- **单帧尺寸：** 64x64 像素
- **立绘尺寸：** 512x768 像素
- **设计图：** 256x256 像素

### 精英怪物
- **单帧尺寸：** 64x64 或 96x96 像素
- **立绘尺寸：** 512x768 像素
- **设计图：** 256x256 像素

### 普通怪物
- **单帧尺寸：** 32x32 或 48x48 像素
- **立绘尺寸：** 256x384 像素
- **设计图：** 128x128 像素

## 动画帧率规范
- **普通动画：** 8-12 帧
- **特殊动画（大招、死亡）：** 12-24 帧
- **帧率：** 12-15 FPS

## 精灵图排列规范
- **排列方式：** 横向排列
- **精灵图总宽度：** 单帧宽度 × 帧数
- **精灵图高度：** 单帧高度

```
精灵图帧排列（横向）：
┌──────┬──────┬──────┬──────┐
│ 帧1  │ 帧2  │ 帧3  │ 帧4  │
└──────┴──────┴──────┴──────┘
  64px   64px   64px   64px

总尺寸：256x64 (4帧 × 64px)
```
