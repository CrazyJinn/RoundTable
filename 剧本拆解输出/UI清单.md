# UI清单

> 本文档列出游戏中的所有UI元素，包括界面、按钮、面板等的设计需求。

---

## 概览

| 分类 | 数量 | 说明 |
|------|------|------|
| 主界面UI | 5 | 主菜单、角色选择等 |
| 游戏内HUD | 5 | 血条、技能栏、弹药/魔法等 |
| 交互UI | 5 | 对话框、商店、背包等 |
| 其他UI | 5 | 加载、暂停、结算等 |
| **总计** | **20+** | 根据需求可扩展 |

## 一、主界面UI

### ui_main_menu - 主菜单

| 字段 | 内容 |
|------|------|
| **UI ID** | ui_main_menu |
| **UI名称** | 主菜单 |
| **UI类型** | 面板 |
| **尺寸规格** | 1920x1080（全屏） |
| **显示时机** | 游戏启动时、退出到主菜单时 |

### 界面描述
游戏启动后的第一个界面，显示游戏标题和主要选项。

### 设计元素
- **背景：** 游戏主题背景图（废土风格/魔法风格切换或融合）
- **标题：** 游戏名称，大字体居中，有特效
- **按钮：** 垂直排列的选项按钮
- **版本信息：** 底部显示版本号
- **版权信息：** 底部显示版权信息

### 包含的子UI
- `ui_btn_start` - 开始按钮
- `ui_btn_settings` - 设置按钮
- `ui_btn_credits` - 制作人员按钮（可选）
- `ui_btn_quit` - 退出按钮

### 美术风格
- **整体：** 简洁大气，突出游戏主题
- **配色：** 废土风格（棕黄灰）或融合风格
- **特效：** 标题有光效/粒子效果

### ui_hp_bar - 血条

| 字段 | 内容 |
|------|------|
| **UI ID** | ui_hp_bar |
| **UI名称** | 血条 |
| **UI类型** | 进度条 |
| **尺寸规格** | 300x30 |
| **位置** | 屏幕左上角或左下角 |
| **显示时机** | 游戏进行时 |

### 界面描述
显示主角当前生命值。

### 设计元素
- **背景：** 暗色背景条
- **血条：** 绿色→黄色→红色渐变（根据血量变化）
- **血量数字：** 当前/最大血量显示
- **低血量警告：** 血量低于30%时闪烁/变红

### 状态
| 血量 | 颜色 |
|------|------|
| **100%-50%** | 绿色 |
| **50%-30%** | 黄色 |
| **30%-0%** | 红色（闪烁警告） |

## 五、UI通用规范

### 5.1 尺寸规范
- **全屏UI：** 1920x1080基准，适配其他分辨率
- **面板UI：** 根据内容调整，保持比例
- **按钮UI：** 统一尺寸和样式
- **图标UI：** 使用统一尺寸的图标

### 5.2 配色规范
| UI类型 | 主色 | 副色 | 强调色 |
|--------|------|------|--------|
| 科技派UI | 深灰 | 黑色 | 深红 |
| 魔法派UI | 紫色 | 深蓝 | 金色 |
| 通用UI | 中性灰 | 浅灰 | 蓝色 |

### 5.3 字体规范
- **标题：** 大字号，粗体，有特效
- **正文：** 中等字号，清晰易读
- **提示：** 小字号，轻量级
- **数字：** 等宽字体，便于对齐

### 5.4 动画规范
- **UI出现：** 淡入/滑入效果
- **UI消失：** 淡出/滑出效果
- **按钮交互：** 悬停/按下有过渡效果
- **进度条：** 平滑过渡

### 5.5 可访问性
- **高对比度：** 重要信息使用高对比度
- **色盲友好：** 不仅用颜色区分状态
- **可调节大小：** UI大小可调节（可选）
- **快捷键支持：** 主要操作支持快捷键
