# 角色清单

> 本文档列出游戏中的所有角色，包括外观描述、动画需求和资源需求。

---

## 概览

| 分类 | 数量 | 说明 |
|------|------|------|
| 主角 | 2 | 科技主角（男）、魔法主角（女） |
| 普通怪（动物类） | 4 | 辐射巨鼠、钢鬃野猪、骨翼鸦、毒液蜘蛛 |
| 精英怪（动物类） | 2 | 裂变熊、双头狼 |
| 普通怪（植物类） | 4 | 藤蔓怪、爆炸孢子菇、吸血藤、酸液喷壶草 |
| 精英怪（植物类） | 2 | 树人守卫、食人花 |
| NPC | **待定** | 根据剧情需求添加 |
| **总计** | **14+** | 不包含NPC |

## 一、主角

### char_player_tech - 科技主角

| 字段 | 内容 |
|------|------|
| **角色ID** | char_player_tech |
| **角色名称** | 科技主角（暂命名：**待定**） |
| **角色类型** | 主角 |
| **性别** | 男 |
| **年龄** | 17岁 |
| **隶属势力** | 骑士团 |
| **设计主题** | 黑白灰外骨骼机甲风格 |

### 外观描述
- **面部：** 年轻男性，短发，眼神坚毅。有战斗留下的轻微伤痕
- **体型：** 中等身材，健壮但不过分魁梧
- **服装：**
  - 黑白灰配色为主
  - 穿着轻型外骨骼机甲
  - 机甲覆盖胸部、肩膀、手臂、腿部
  - 服装有科技派的标识
- **武器：** 初期为制式步枪，后期可更换
- **配色：**
  - 主色：黑色、深灰色
  - 副色：白色（机甲细节）
  - 强调色：骑士团标识色

### 动画需求

| 动画名称 | 说明 | 帧数估算 |
|----------|------|----------|
| **待机（Idle）** | 站立，轻微呼吸动作，偶尔检查武器 | 8-12帧 |
| **移动（Move）** | 8方向移动，机甲助力奔跑 | 每方向8-12帧 |
| **翻滚（Roll）** | 翻滚躲避动作，机甲提供额外速度 | 12-16帧 |
| **普通攻击（Attack）** | 举枪射击，有后坐力动作 | 8-12帧 |
| **装弹（Reload）** | 更换弹夹的动作 | 16-24帧 |
| **技能1（Skill1）** | 根据技能设计确定 | 待定 |
| **技能2（Skill2）** | 根据技能设计确定 | 待定 |
| **终极技能（Ultimate）** | 强力技能释放动作 | 待定 |
| **受伤（Hurt）** | 受击反馈，身体后仰 | 4-6帧 |
| **死亡（Death）** | 倒地死亡 | 16-24帧 |

### 所需资源
- **精灵图格式：** Sprite Sheet（建议512x512或更大）
- **朝向：** 至少8方向（上、下、左、右及四个斜方向）
- **缩放：** 游戏中约64x64像素（根据最终分辨率调整）
- **文件格式：** PNG（透明背景）

### 特殊效果
- **射击效果：** 枪口火焰、子弹轨迹、命中火花
- **翻滚效果：** 残影、尘土
- **技能效果：** 根据技能设计添加

### 7.1 精灵图规范

- **Sprite Sheet布局：** 横向或纵向排列，均匀分布
