# 场景清单

> 本文档列出游戏中的所有场景，包括背景、角色、UI、音效、交互逻辑和剧情内容的详细需求。

---

## 概览

| 分类 | 数量 | 说明 |
|------|------|------|
| 大地图场景 | 3 | 废土、变异森林、城市废墟 |
| 城镇场景 | 6 | 科技派3个 + 魔法派3个 |
| 战斗场景 | 3+ | 对应三大区域，可扩展 |
| **总计** | **12+** | 根据关卡设计可增加 |

## 一、大地图场景

### scene_world_001 - 废土

| 字段 | 内容 |
|------|------|
| **场景ID** | scene_world_001 |
| **场景名称** | 废土 |
| **场景类型** | 大地图 |
| **场景描述** | 核战后荒凉的废土区域。黄褐色调为主，大地干裂，黄沙漫天。远处可见倒塌的摩天大楼残骸，近处散落着废弃的车辆、生锈的金属构件。偶尔可见核战留下的辐射坑洞，散发着危险气息。天空浑浊，沙尘暴随时可能来袭。 |
| **美术风格** | 废土风格，黄棕褐色调，荒凉破败感 |
| **所需背景** | 大尺寸可平铺的废土背景图（建议4096x4096或更大），包含：远处建筑残骸、近处废弃物体、地面裂纹、沙尘效果 |
| **所需角色** | 主角、可选的巡逻NPC |
| **所需UI** | 小地图（显示当前区域和任务标记）、任务追踪面板、交互提示 |
| **所需音效** | 荒凉风的背景音乐、风声音效、沙尘声、偶尔的金属碰撞声 |
| **交互逻辑** | <ul><li>鼠标点击地面移动主角</li><li>与场景物体（废弃车辆、残骸）可交互搜索物资</li><li>与NPC对话可接取任务</li><li>进入特定区域触发战斗场景</li></ul> |
| **剧情内容** | <ul><li>新手教学区域</li><li>初始任务触发点</li><li>探索可发现隐藏物资点</li></ul> |
| **特殊元素** | 辐射区域（进入扣血）、避难所入口（可选剧情） |
| **天气效果** | 沙尘暴（能见度降低）、强烈阳光 |
| **光照** | 昏暗的自然光，夕阳效果最佳 |

### scene_world_002 - 变异森林

| 字段 | 内容 |
|------|------|
| **场景ID** | scene_world_002 |
| **场景名称** | 变异森林 |
| **场景类型** | 大地图 |
| **场景描述** | 核辐射使森林发生严重变异。树木扭曲成诡异形状，树干上散发着幽绿和紫色的光芒。植物异常巨大，藤蔓遍布。地面覆盖着发光的苔藓和菌类。空气中弥漫着诡异的雾气。远处可听到不明生物的叫声。 |
| **美术风格** | 诡异神秘风格，绿紫色调为主，发光植物效果 |
| **所需背景** | 大尺寸可平铺的变异森林背景图，包含：扭曲的巨大树木、发光植物、藤蔓、雾气效果、奇异的花朵 |
| **所需角色** | 主角、野生怪物（藤蔓怪、爆炸孢子菇等） |
| **所需UI** | 小地图、任务追踪面板、交互提示、危险区域警告 |
| **所需音效** | 神诡的森林背景音乐、虫鸣声、树叶沙沙声、不明生物叫声 |
| **交互逻辑** | <ul><li>鼠标点击移动</li><li>与植物可交互（部分可采集，部分有攻击性）</li><li>进入危险区域自动触发战斗</li><li>发现隐藏道具点</li></ul> |
| **剧情内容** | <ul><li>探索魔法派起源的相关线索</li><li>遭遇变异植物怪物</li><li>发现秘密研究设施遗迹</li></ul> |
| **特殊元素** | 剧毒沼泽（进入扣血/减速）、发光花朵（可采集）、古代遗迹 |
| **天气效果** | 诡雾（能见度降低）、荧光孢子飘浮 |
| **光照** | 昏暗，主要依靠植物发光提供光源 |
